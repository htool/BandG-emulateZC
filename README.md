# Emulate ZC1/2

## Usage

To start the emulation e.g. on ID 19 use:
$ DEBUG=emulate node emulate.js ZC2 19

## What works
 - On console you can use the following keys:

space/return/enter = knob push
d = display
pageup = zoom out
pagedownA = zoom in
p = power
m = menu
up = up
down = down 
left = left
right = right
escape = pages
